Asel Rasimgil 24040224 Tic Tac Toe Projesi
# Proje Açıklaması
Bu proje, Python kullanılarak yapılmış basit bir Tic Tac Toe oyunudur.
Oyun 3x3'lük bir tablo üzerinde oynanır ve bilgisayar 'X', kullanıcı 'O' ile oynar.
İlk hamle her zaman bilgisayara aittir.

Oyun sırasında kullanıcı geçerli kareyi seçer (1-9 arasında). 
Oyun sonunda kazanma, kaybetme veya berabere durumu kontrol edilir.
